# Geumcheon-Meat
금천미트 구매후기를 바탕으로한 장바구니 분석
구매후기 데이터를 얻은 금천미트
금천미트는 B2B 온라인몰으로써 회원들의 재구매율이 상당히 높다.
따라서 재구매 내역을 바탕으로 구매이력을 추정할 수 있었고
이러한 구매이력을 바탕으로 장바구니분석(상품연관분석)을 진행하였다.
